# Gemini CLI キーボードショートカット

Gemini CLI には、入力の編集、履歴のナビゲート、UI の制御のためのデフォルトのキーボードショートカットが付属しています。このリファレンスを使用して、利用可能な組み合わせを学習してください。

#### 基本的なコントロール

| アクション                               | キー      |
| -------------------------------------------- | --------- |
| 現在の選択または選択肢を確認します。     | `Enter`   |
| ダイアログを閉じるか、現在のフォーカスをキャンセルします。 | `Esc`     |

#### カーソル移動

| アクション                                    | キー                   |
| ----------------------------------------- | ---------------------- |
| カーソルを行の先頭に移動します。 | `Ctrl + A`<br />`Home` |
| カーソルを行の末尾に移動します。   | `Ctrl + E`<br />`End`  |

#### 編集

| アクション                                           | キー                                      |
| ------------------------------------------------ | ----------------------------------------- |
| カーソルから行の末尾まで削除します。   | `Ctrl + K`                                |
| カーソルから行の先頭まで削除します。 | `Ctrl + U`                                |
| 入力フィールドのすべてのテキストをクリアします。               | `Ctrl + C`                                |
| 前の単語を削除します。                        | `Ctrl + Backspace`<br />`Cmd + Backspace` |

#### 画面コントロール

| アクション                                       | キー       |
| -------------------------------------------- | ---------- |
| ターミナル画面をクリアし、UI を再描画します。 | `Ctrl + L` |

#### スクロール

| アクション                   | キー                 |
| ------------------------ | -------------------- |
| コンテンツを上にスクロールします。       | `Shift + Up Arrow`   |
| コンテンツを下にスクロールします。     | `Shift + Down Arrow` |
| 先頭までスクロールします。       | `Home`               |
| 末尾までスクロールします。    | `End`                |
| 1ページ上にスクロールします。   | `Page Up`            |
| 1ページ下にスクロールします。 | `Page Down`          |

#### 履歴と検索

| アクション                                       | キー                  |
| -------------------------------------------- | --------------------- |
| 履歴の前のエントリを表示します。          | `Ctrl + P (no Shift)` |
| 履歴の次のエントリを表示します。              | `Ctrl + N (no Shift)` |
| 履歴の逆方向検索を開始します。        | `Ctrl + R`            |
| 選択した逆方向検索の一致を挿入します。    | `Enter (no Ctrl)`     |
| 逆方向検索中に提案を受け入れます。 | `Tab`                 |

#### ナビゲーション

| アクション                           | キー                                        |
| -------------------------------- | ------------------------------------------- |
| リスト内の選択を上に移動します。      | `Up Arrow (no Shift)`                       |
| リスト内の選択を下に移動します。    | `Down Arrow (no Shift)`                     |
| ダイアログオプション内で上に移動します。   | `Up Arrow (no Shift)`<br />`K (no Shift)`   |
| ダイアログオプション内で下に移動します。 | `Down Arrow (no Shift)`<br />`J (no Shift)` |

#### 提案と補完

| アクション                                  | キー                                               |
| --------------------------------------- | -------------------------------------------------- |
| インラインの提案を受け入れます。           | `Tab`<br />`Enter (no Ctrl)`                       |
| 前の補完オプションに移動します。 | `Up Arrow (no Shift)`<br />`Ctrl + P (no Shift)`   |
| 次の補完オプションに移動します。     | `Down Arrow (no Shift)`<br />`Ctrl + N (no Shift)` |
| インラインの提案を展開します。            | `Right Arrow`                                      |
| インラインの提案を折りたたむ。          | `Left Arrow`                                       |

#### テキスト入力

| アクション                               | キー                                                                                        |
| ------------------------------------ | ------------------------------------------------------------------------------------------- |
| 現在のプロンプトを送信します。           | `Enter (no Ctrl, no Shift, no Cmd, not Paste)`                                              |
| 送信せずに改行を挿入します。 | `Ctrl + Enter`<br />`Cmd + Enter`<br />`Paste + Enter`<br />`Shift + Enter`<br />`Ctrl + J` |

#### 外部ツール

| アクション                                         | キー       |
| ---------------------------------------------- | ---------- |
| 現在のプロンプトを外部エディターで開きます。 | `Ctrl + X` |
| クリップボードから貼り付けます。                      | `Ctrl + V` |

#### アプリコントロール

| アクション                                                            | キー       |
| ----------------------------------------------------------------- | ---------- |
| 詳細なエラー情報を切り替えます。                                | `F12`      |
| 完全な TODO リストを切り替えます。                                        | `Ctrl + T` |
| IDE コンテキストの詳細を切り替えます。                                       | `Ctrl + G` |
| Markdown レンダリングを切り替えます。                                        | `Cmd + M`  |
| ターミナルが代替バッファを使用しているときにコピーモードを切り替えます。 | `Ctrl + S` |
| 高さ制限のある応答を展開して追加の行を表示します。    | `Ctrl + S` |
| シェルと Gemini 入力の間でフォーカスを切り替えます。                  | `Ctrl + F` |

#### セッションコントロール

| アクション                                       | キー       |
| -------------------------------------------- | ---------- |
| 現在のリクエストをキャンセルするか、CLI を終了します。  | `Ctrl + C` |
| 入力バッファが空のときに CLI を終了します。 | `Ctrl + D` |

## 追加のコンテキスト固有のショートカット

- `Ctrl+Y`: ツール呼び出しの YOLO (自動承認) モードを切り替えます。
- `Shift+Tab`: 自動編集 (自動承認編集) モードを切り替えます。
- `Option+M` (macOS): Option+M で `µ` を入力すると、`Cmd+M` と同様に Markdown レンダリングも切り替わります。
- 空のプロンプトで `!`: シェルモードに入るか終了します。
- `\` (行末) + `Enter`: シングルラインモードを終了せずに改行を挿入します。
- `Ctrl+Delete` / `Meta+Delete`: カーソルの右側の単語を削除します。
- `Ctrl+B` または `Left Arrow`: テキストの編集中にカーソルを1文字左に移動します。
- `Ctrl+F` または `Right Arrow`: カーソルを1文字右に移動します。埋め込みシェルが接続されている場合でも、`Ctrl+F` はフォーカスを切り替えます。
- `Ctrl+D` または `Delete`: カーソルのすぐ右側の文字を削除します。
- `Ctrl+H` または `Backspace`: カーソルのすぐ左側の文字を削除します。
- `Ctrl+Left Arrow` / `Meta+Left Arrow` / `Meta+B`: 1単語左に移動します。
- `Ctrl+Right Arrow` / `Meta+Right Arrow` / `Meta+F`: 1単語右に移動します。
- `Ctrl+W`: カーソルの左側の単語を削除します（`Ctrl+Backspace` / `Cmd+Backspace` に加えて）。
- `Ctrl+Z` / `Ctrl+Shift+Z`: 最新のテキスト編集を元に戻すか、やり直します。
- `Meta+Enter`: 現在の入力を外部エディターで開きます（`Ctrl+X` のエイリアス）。
- `Esc` を素早く2回押す: 現在の入力バッファをクリアします。
- 選択ダイアログ内の`数字キー (1-9, 複数桁)`: 番号付きのラジオオプションに直接ジャンプし、確認します。
