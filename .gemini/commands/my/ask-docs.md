# Ask Docs

**呼び出し方法**: `/my:ask-docs <質問>`

**説明**  
ローカルのドキュメントフォルダ (`docs/`, `docs_j/`, `.gemini/`) を検索し、ユーザーの質問に回答します。
参照したファイルへのパスを明記して回答を生成します。

## ミッション: ローカルドキュメントから質問に回答する

あなたのタスクは、現在のワークスペース内の関連ドキュメントを検索・閲覧し、ユーザーの質問に答えることです。

**対象ディレクトリ:**
- `docs/`
- `docs_j/` (日本語ドキュメント)
- `.gemini/`

### ワークフロー:

1.  **ドキュメントの検索**:
    *   `glob` または `list_directory` を使用して、ユーザーの質問 (`{{args}}`) に対する回答が含まれていそうなファイルを対象ディレクトリから検索します。
    *   クエリ内のキーワードと一致するファイル名を優先します。

2.  **コンテンツの読み込み**:
    *   `read_file` を使用して、最も有望なファイルの内容を読み込みます。

3.  **回答の合成**:
    *   ドキュメント内で見つかった情報**のみ**に基づいて回答します。
    *   **引用必須**: 情報の出処となるファイルパスを `[Source: path/to/file.md]` の形式で明記する必要があります。
    *   情報が見つからない場合は、その旨を明確に伝えてください。答えを捏造しないでください。
