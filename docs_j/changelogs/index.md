# Gemini CLI の変更履歴

このページでは Gemini CLI の主なリリースハイライトを紹介します。

## v0.15.0 (2025-11-03)
- UI をスクロール可能にしてマウス操作を追加し、ちらつきのないヘッダーと安定した入力欄を提供。
- Arize、Chronosphere、Transmit などのパートナー拡張機能を追加。
- Todo プランニングを導入し、複雑な要求を一覧で管理。
- GitHub 拡張の無効化、拡張の再起動コマンド、Angular ワークフローの改善、ローカル拡張の検証機能など。

## v0.12.0 (2025-10-27)
- Hugging Face / Monday.com / Data Commons などパートナー拡張を拡充。
- モデル選択・ルーティングと Codebase Investigator サブエージェントを追加。
- `/extension explore`、圧縮閾値の調整、API キー認証ダイアログ、順次承認の改善。

## v0.11.0 (2025-10-20)
- Jules 拡張、JSONL ストリーミング、Markdown 表示トグル、入力履歴編集などを追加。
- JSON Web Fetch、ヘッドレスで MCP コマンド実行、非推奨フラグ削除。

## v0.10.0 (2025-10-13)
- インタラクティブシェル、Alt キーショートカット、Telemetry Diff、品質改善。

## v0.9.0 (2025-10-06)
- インタラクティブシェル操作（`vim`, `rebase -i` など）、プレリリース拡張のインストール、テンプレート不要の拡張生成、OpenTelemetry 対応、`/memory list` などを導入。

## v0.8.0〜v0.5.0
- 拡張機能の一般公開、CLI ドキュメント刷新、非対話モードの `--allowed-tools`、ステータス表示、FastMCP 統合、非対話的なプロンプト、トレースや JSON モードの強化など。
- 通信・入力の改善、`write_todos` や引用、カスタムコマンドの `@{path}`、Gemini 3 Pro への容量増強など。

手元のバージョンで変更内容を確認するには、該当 PR や公式ブログをご参照ください。
