# Change Mode

**呼び出し方法**: `/my:ch <plan|edit>`

**説明**  
Plan Mode と Edit Mode を切り替え、エージェントの振る舞いを変更します。
引数でモードを指定します。

Plan Mode なら Gemini のモデルを Flash に、Edit Mode なら Pro に切り替える運用（手動または設定）とセットで使用します。

### 1. Plan Mode (指定: plan, p)
- **役割**: 要件定義、計画、ドキュメント作成、議論。
- **制約**: **コードの生成・修正は禁止**。ファイル操作は `.md` や `.txt` のみ許可。
- **振る舞い**: ユーザーの意図を汲み取り、計画を練ることに集中する。

### 2. Edit Mode (指定: edit, e)
- **役割**: 実装、コーディング、リファクタリング。
- **制約**: コードの生成・修正が可能。
- **振る舞い**: 設計に基づき、具体的で高品質なコードを記述する。
