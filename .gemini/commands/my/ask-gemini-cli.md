# Ask Gemini CLI

**呼び出し方法**: `/my:ask-gemini-cli`

**説明**  
ワークスペース内のドキュメントをコンテキストとして読み込んだ状態で、Gemini CLI に関する質問に対話形式で回答します。
実行すると、まずユーザーに質問内容を尋ねます。

## 指示

1.  まずユーザーに「Gemini CLIについて何か質問はありますか？」と尋ねて開始します（まだ質問には答えません）。
2.  ユーザーの応答を待ちます。
3.  ユーザーが質問したら、内容を理解したことを確認し、上記の「Documentation Context」に含まれる情報**のみ**を使用して回答します。
4.  可能であればソースファイル名を引用してください（例：「`docs/cli/settings.md`によると...」）。
5.  情報がコンテキストに存在しない場合は、その旨を明確に述べてください。
6.  必要に応じて会話を続け、詳細を確認したり提供したりします。
